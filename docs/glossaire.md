# Termes et définitions

| Terme | Définition |
|---|---|
| Bourse | Événement d’échange/vente organisé par ALPE avec dépôt d’articles, vente, reversement. |
| Déposant | Personne qui confie des articles à vendre lors d’une bourse. |
| Bénévole | Membre organisateur qui gère la bourse (préparation, encaissement, clôture). |
| Article | Bien individuel proposé à la vente (taille, catégorie, prix). |
| Lot | Regroupement d’articles sous un même dépôt/déposant. |
| Étiquette | Code (ex: QR/Code‑barres) identifiant article/lot pour scannage. |
| Vente | Transaction associant un article vendu et un paiement. |
| Commission | Part de l’association prélevée sur le prix de vente. |
| Reversement | Somme due au déposant après déduction des commissions/frais. |
| Session de bourse | Édition spécifique (dates, lieu, règles tarifaires). |

# Règles de vocabulaire

- Utiliser les termes ci‑dessus de manière cohérente dans toutes les specs.
- Ajouter tout nouveau terme ici avant usage dans d’autres documents.
