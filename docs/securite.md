# Authentification & Autorisation

- Rôles initiaux: `deposant`, `benevole`, `admin`.
- Séparation stricte des permissions (moindre privilège).

# Données personnelles (RGPD)

- Base légale: intérêt légitime de l’organisation + consentement pour communications.
- Droits: accès, rectification, suppression; politique de conservation (limitation dans le temps).

# Journalisation & traçabilité

- Journaux d’accès (minimaux), événements sensibles (paiements, clôtures).

# Menaces & contrôles

- Anti‑fraude étiquettes (unicité, signature/QR), validation des montants.
- Protection des interfaces caisse (session courte, verrouillage).
