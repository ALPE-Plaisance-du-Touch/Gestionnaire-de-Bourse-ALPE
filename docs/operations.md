# SLOs (ébauche)

- Disponibilité bourse (fenêtre événement): 99.5%.
- Latence encaissement (p95): ≤ 3 s.

# Observabilité

- Métriques: ventes/minute, échecs scan, temps encaissement.
- Journaux: erreurs applicatives, anomalies scannage.

# Runbooks

- Incident réseau: bascule offline (si supporté), file d’attente, resynchronisation.
- Erreur étiquette: re‑génération, invalidation, suivi.
